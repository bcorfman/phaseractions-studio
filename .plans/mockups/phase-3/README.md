# Phase 3 Mockups (Polished Editor)

These are lightweight SVG wireframes for Phase 3 behaviors in `.plans/direct-manipulation-canvas-editing.md`.

- `phase3-01-marquee-multiselect.svg`: marquee multi-select, selection toolbar, pin affordance
- `phase3-02-regroup-ungroup.svg`: regroup/ungroup UX (context menu + selection actions)
- `phase3-04-undo-redo-snap-playmode.svg`: undo/redo batching, grid snapping toggle, play/edit mode toggle
- `phase3-05-dragdrop-group-membership.svg`: drag sprites onto formations to add; drag members into Sprites to remove
- `phase3-06-cursor-context-menu.svg`: right-click opens selection actions menu at cursor position
- `phase3-07-topright-selection-actions.svg`: optional top-right selection actions cluster (Create Group / Add to Group…)
- `phase3-08-convert-group-layout-submenu.svg`: Convert Group Layout submenu for group selection
